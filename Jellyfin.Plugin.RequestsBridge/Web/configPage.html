Requests Bridge Settings

The base URL of your Jellyseerr instance (e.g., http://192.168.1.100:5055)
Optional: A Jellyseerr URL reachable via Tailscale for remote devices (e.g., http://100.x.y.z:5055). This is returned via /plugins/requests/tailscale/apibase.
Optional: Your Jellyfin URL reachable via Tailscale for remote devices (e.g., 100.99.17.85:8096). This is returned via /plugins/requests/tailscale/jellyfinbase.
Your Jellyseerr API key. Find it in Jellyseerr under Settings → General. The key can be retrieved via /plugins/requests/apikey (for Android TV apps).

Performance Settings

Enable this to fix slow loading of Collections. This disables watch status and progress indicators for Collections, which significantly improves loading performance. Based on jellyfin-plugin-disable-user-data.

API Endpoints

Get Jellyseerr Base URL: GET /plugins/requests/apibase

Get Tailscale Jellyseerr URL: GET /plugins/requests/tailscale/apibase

Get Tailscale Jellyfin URL: GET /plugins/requests/tailscale/jellyfinbase

Get API Key: GET /plugins/requests/apikey

These endpoints return the configured URLs and API key. Useful for Android TV app.
