Requests Bridge Settings

The base URL of your Jellyseerr instance (e.g., http://192.168.1.100:5055)
Your Jellyseerr API key. Find it in Jellyseerr under Settings → General. The key can be retrieved via /plugins/requests/apikey (for Android TV apps).

API Endpoints

Get Jellyseerr Base URL: GET /plugins/requests/apibase

Get API Key: GET /plugins/requests/apikey

This endpoint returns the configured Jellyseerr URL and API key. Useful for Android TV app.
